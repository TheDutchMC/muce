MUCE By Tobias de Bruijn

Muce stands for Multi User Coding Environment. It's based on VSCode by Microsoft. The frontend is written in TypeScript and SCSS. The backend is written in Java.

Muce uses Docker and Traefik as a backbone for making this all work. When you log in, the API asks Docker to create a container. This container has labels attached, which Traefik picks up. Traefik then adds a route, so that when you load the workspace, you get the correct container.

Muce also comes with a build-in webserver for viewing what you've made. PHP 7.4 is installed.

This frontend runs on GitHub pages, the backend is hosted on my own infrastructure.

Backend dependencies:

  • org.springframework:spring-web

  • org.springframework.boot:spring-boot-starter-web

  • mysql:mysql-connector-java

The container has the following software installed:

  • Make

  • GCC

  • G++

  • OpenJDK 11

  • Python3

  • Python3 Pip

  • Unzip

  • wget

  • Gradle 6.8.1

The following extensions are packaged with the container, and can be found in /ext/:

  • MS-VSCode cpptools

  • MS-VSCode TypeScript

  • Naco Siren Gradle Language

  • RedHat Java Language

  • Richardwillis Gradle Language

  • VSCode Java Debug

  • VSCode Java Pack
